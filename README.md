# 🎯 تطبيق التسويق الاحترافي
## Professional Marketing App - Offline/Online

> تطبيق ويب احترافي متقدم لتسويق المنتجات على منصات التواصل الاجتماعي مع إمكانية العمل بدون إنترنت

### ✨ المميزات الرئيسية:

#### 🔍 كشف الأسعار الذكي
- كشف أكثر من 40 صيغة مختلفة للأسعار
- دعم العملات: يمني (قديم/جديد)، سعودي، دولار
- تحويل تلقائي بين العملات
- إضافة نسبة مئوية محددة تلقائياً
- استبدال الأسعار في النص والصور

#### 📝 معالجة النصوص الذكية
- إعادة صياغة احترافية متعددة
- ترتيب ذكي للفقرات
- تلوين تلقائي مختلف كل مرة
- اقتراحات تسويقية ذك��ة فورية
- إضافة أيقونات حسب المحتوى
- تصحيح تلقائي للأخطاء
- مسافات ذكية بين الفقرات

#### 🖼️ معالجة الصور
- إضافة النصوص والأسعار على الصور تلقائياً
- تحكم كامل بالمس (سحب وتحريك)
- شريط عائم للتحكم بكل نص
- تحكم بالألوان والخطوط والأحجام
- قص وتعديل الصور
- رموز إيموجي متقدمة

### 🛠️ التقنيات المستخدمة:
React 18 + TypeScript + Tailwind CSS + PWA + IndexedDB
